# tic-tac-toe
Two users play a game of tick-tack-toe on the same computer

## Installation and use

```
$ git clone https://github.com/amirphp7/tic-tac-toe.git
```
```
$ cd tic-tac-toe
```
### Unix / Linux
```
$ yarn
```
```
$ yarn start
```
### Windows
```
$ npm install
```
```
$ npm start
```
Runs the app in the development mode.<br>
**  http://localhost:3000 **
